# 🔊 Text to Speech Converter

A simple and interactive web application that converts text into speech using the browser's Speech Synthesis API. Users can select different voices and listen to the spoken output in real time.

🚀 Features

*  Convert text into speech instantly
*  Multiple voice options (different languages & accents)
*  Dynamic voice selection
*  Fast and browser-based (no installation needed)
*  Simple and responsive UI


##Tech Stack

* HTML
* CSS
* JavaScript (Speech Synthesis API)


How It Works

1. Enter text in the textarea
2. Select a voice from the dropdown
3. Click the **Speak** button
4. The browser will read the text aloud
